    校园网设计实验

    1.创建vlan

    在conf t 模式下

    swithch(config) vlan 2 //创建vlan

    swithch(config-vlan) name v2 //为创建的vlan命名

    2.给端口分配vlan

    分配接入端口。也就是分配vlan

    swithch(config) int fa0/1 //进入对应端口

    swithch(config-if) switchport mode access //设置端口协议

    swithch(config-if) switchport access vlan 2 //为端口设置vlan

    共享端口。供不同的vlan通过交换

    swithch(config) int fa0/1 //进入对应端口

    swithch(config-if) switchport trunk encapsulation dot1q //对于三层交换机设置端口必须有这一步

    swithch(config-if) switchport mode trunk //设置端口协议

    swithch(config-if) switchport trunk allowed vlan 2,3 //设置允许通过的vlan

    3.为对应vlan设置ip地址

    (config) int vlan 2

    (config-if) ip addr 192.1.2.254 255.255.255.0

    4.启动三层交换机路由功能

    (config) ip routing

    总结：该实验中，二层交换机进行创建vlan，给端口分配vlan。而三层交换机则要创建vlan，给端口分配vlan，同时为对应vlan设置ip地址，最后启动交换机路由功能

    三层交换机动态路由配置实验

    需要将两条两路聚合成一条

    1.创建端口通道

    switch(config) # int range fa0/3-fa0/5 //进入要聚合的端口

    switch(config-if-range) # channel-group 1 mode active //创建编号为1的端口通道，分配给上述端口，并激活。（端口通道编号针对于本交换机而言，即不同交换机都是从1开始的）

    switch(config-if-range) # channel-protocol lacp //指定聚合协议为lacp

    switch(config-if-range) # exit

    switch(config) # port-channel load-balance src-dst-mac 指定分发策略

    2.给端口通道分配vlan

    接入端口

    swithch(config) int port-channel 1 //进入聚合端口 对应的编号

    swithch(config-if) switchport mode access //设置端口协议

    swithch(config-if) switchport access vlan 2 //为端口设置vlan

    共享端口

    swithch(config) int port-channel 2 //进入对应端口

    swithch(config-if) switchport trunk encapsulation dot1q //对于三层交换机设置端口必须有这一步

    swithch(config-if) switchport mode trunk //设置端口协议

    swithch(config-if) switchport trunk allowed vlan 2,3 //设置允许通过的vlan

    3.配置ospf

    swithch(config) #router ospf 07 //07 为标识符 ，一个线路中，每个都要不一样

    network 192.1.2.0 0. 0. 0.255 area 1 // 相连的网络号 子网掩码的反码

    企业网路由项和NAT配置

    打开路由器端口，为端口分配IP地址

    int fa0/0

    no shut

    ip addr 192.168.1.254 255.255.255.0

    ospf路由

    router ospf 01

    Router8，Router9，Router10

    1. 打开路由器端口，为端口分配IP地址

    2. ospf路由

    此时左边因为是内部网络所以访问外部网络无法得到响应

    所以需要net地址转换

    1.建立全球ip地址池和私有地址的关联

    Router(config) # access-list 1 permit 192.168.1.0 0.0.0.255 //创建一个编号为1的标准分组过滤器允许转发192.168.1.0的ip分组。 这里规定了nat的私有地址范围

    Router(config) # ip nat pool a1 193.1.4.1 193.1.4.13 netmask 255.255.255.240// 定义全球ip地址池

    Router(config) # ip nat inside source list 1 pool a1 //私有地址和全球ip地址池绑定

    2.创建静态地址转换项

    Router(config) # ip nat inside source static 192.168.2.3 193.1.4.14 //内部地址和全球地址绑定

    3.配置内部网络连接外部网络接口

    Router(config) # int fa0/0

    Router(config-if) # ip nat inside //将该接口配置为内部网络接口

    Router(config) # int fa1/0

    Router(config-if) # ip nat outside //将该接口配置为外部网络接口

    4.静态路由项配置

    ip route 193.1.4.0 255.255.255.240 193.1.1.1 //193.1.4.0 255.255.255.240为目标网络。193.1.1.1为下一跳的地址。

    分组过滤器配置实验

    分组过滤器

    Router(config) # access-list 101 permit tcp 193.1.4.0 0.0.0.15 host 193.1.5.3 eq www //建立编号为101的过滤规则，允许193.1.4.0对193.1.5.3转发端口号为www的报文

    Router(config) # access-list 101 permit ospf host 193.1.1.1 any

    Router(config) # access-list 101 deny ip any any

    定义检测机制

    Router(config) # ip inspect name in1 http

    Router(config) # ip inspect name in1 tcp

    作用接口

    Router(config) # int fa0/0

    Router(config) # ip access-group 101 in

    Router(config) # ip access-group 102 out

    Router(config) # ip inspect in1 in

    isp 网络设计

    1.配置串行接口

    在2811路由器中添加serial接口wic-2T。

    int Serial0/3/0

    clock rate 4000000 //dce一侧需要设置定时器

    no shut

    ip addr 192.1.3.1 255.255.255.252

    encapsulation ppp //配置协议

    ppp authtication chap //建立连接需要验证身份

    2.配置鉴别信息

    hostname Router1 //设置本机的身份

    username Router2 password 1234 //指定对方路由器的身份为router2，共享密码要相同都为1234

    3.配置RIP

    router rip

    version 2

    no auto-summary

    network 192.1.1.0

    network 192.1.3.0
